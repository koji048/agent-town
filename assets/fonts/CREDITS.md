# Caption fonts

All fonts here are Thai/Latin typefaces from [Google Fonts](https://fonts.google.com),
licensed under the **SIL Open Font License 1.1** (see `OFL.txt`). The Caption
Review Studio discovers every `.ttf`/`.otf` in this folder automatically — drop a
new OFL font in and it appears in the picker.

## Families

Originally bundled: Anuphan, Kanit, Mitr, Prompt, Sarabun.

Added (Google Fonts, SIL OFL — mostly by Cadson Demak):
Athiti, Bai Jamjuree, Chakra Petch, Charm, Charmonman, Chonburi, Fahkwang,
Itim, K2D, Kodchasan, Krub, Maitree, Mali, Niramit, Pattaya, Pridi, Sriracha,
Taviraj, Thasadith, Trirong.

Each family's full copyright notice is carried inside its own font file (the
`name` table) and covered by the SIL OFL in `OFL.txt`.
