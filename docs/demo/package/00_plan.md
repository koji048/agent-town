# Production Brief: 5 เทคนิค AI เร่งคอนเทนต์

**Core Message:** มือใหม่ก็ใช้ AI ทำคอนเทนต์ได้เร็วขึ้น 3 เท่า ด้วย 5 เทคนิคที่ทำตามได้ทันที

**Hook Angle:** เปิดด้วยปัญหาที่เจ็บ — "ทำคอนเทนต์ 1 ตัวใช้เวลา 3 ชม.?" แล้วสัญญาผลลัพธ์ทันที "5 เทคนิคนี้ลดเหลือ 30 นาที" (pain → promise, ตัวเลขชัดในคำแรก)

## Crew Instructions

- **Researcher:** หาเครื่องมือ AI ฟรี/ราคาถูก 5 ตัวที่ครีเอเตอร์ไทยมือใหม่เข้าถึงได้จริง (script, caption, thumbnail, edit, hashtag) พร้อมเคสตัวอย่างผลลัพธ์จริง 1 อันต่อเทคนิค

- **Scriptwriter:** เขียนสคริปต์ 45 วิ โครงสร้าง hook (3s) → 5 เทคนิค (6s ต่ออัน) → CTA (2s) ใช้ประโยคสั้น ภาษาพูด แทรก English hook คำเดียว เช่น "Game changer" เพื่อความติดหู

- **Editor:** ตัดต่อจังหวะเร็ว เปลี่ยนภาพทุก 2-3 วิ, ใช้ text overlay ตัวเลข (1/5, 2/5...) กำกับทุกเทคนิค, ใส่ caption ภาษาไทยเต็มจอสำหรับ sound-off viewing

- **Publisher:** โพสต์ช่วง 19:00-21:00, แคปชั่นเปิดด้วยคำถาม + hashtag ผสม #AIContent #ครีเอเตอร์มือใหม่ #ContentCreatorTips ติด pin comment ลิงก์เครื่องมือทั้ง 5
